# 90、Qt5实现一个文件对话框

需要  #include <QFileDialog>

```
QString file_name=QFileDialog::getOpenFileName(this,"请选择需要打开的文件：",".","*.txt *.png");  //打开文件对话框
        //参数1 父控件
        //参数2 标题
        //参数3  默认路径
        //参数4 过滤文件格式
        //返回值  文件全路径---"D:/ss/注意事项.txt"
     qDebug()<<file_name;
```
